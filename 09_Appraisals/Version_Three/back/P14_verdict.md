# Page 14 — The Verdict

This is the page the seller turns to first if they pick the document up two weeks later. It must be self-contained — readable as the report's conclusion without re-reading the spreads.

The emotion this page must produce: **decision.** "We should list with Fields."

## Designer notes

- **No image.** This page must feel as confident as a contract — calm, declarative, inarguable.
- **The opening assessment uses the largest body type in the document (16pt cap-serif).** This is deliberate. The seller's eye should fall on this passage and read it slowly. The closing three-line cadence ("That category is scarce. / Scarcity changes buyer behaviour. / Behaviour determines price.") is the rhetorical climax of the entire report — it sums the thesis from page 02 in three lines.
- **The three-row recommendation block sits in lower-third of the page** with generous spacing. Each row's label is small-caps copper. Each row's value is the same regular cap-serif as the body, slightly larger. The price range gets visual prominence by virtue of the dollar signs and copper colour treatment.
- **Listing window is preserved here** — the only place timing appears in the V3 document. V2 had a dedicated Spread 10 for timing; V3 collapses it to one line. Production must pull the latest seasonality data at issue time and update if the window shifts.
- **The signature is the only place Will's name appears in the body of the document until the closing note.** It signals: this is a personal recommendation, not an algorithmic output.
- **No bullets. No tables. No second-tier "alternative scenarios."** A verdict is a verdict.

---

## What this page must achieve

The seller must finish this page with no remaining questions.

The four operational unknowns sellers carry into an appraisal — *what's it worth, when should I sell, how should it be marketed, who's running it* — are all answered on this single page.

The page is also designed to be photographed and shared. A seller showing their partner the appraisal will most often turn to *this page* to make the case. Every line on this page has to survive being read alone, without the surrounding spreads.
