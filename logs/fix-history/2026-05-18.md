## [COVERAGE-AUDIT-004] Floor plans extracted from v2 image arrays via GPT-4o-mini vision ‚Äî 18:08 AEST

**Symptom:** Floor-plan coverage on cadastral records was 393/11,953 = 3.3% in Robina, with the prevailing assumption that floor plans live only on listing pages (not Domain property-profile pages). Will pointed out the assumption was wrong: Domain's profile page for 16 Tullamarine Dr embeds the floor plan as image 29 of 29 in the `media({"categories":["IMAGE"]})` array, untagged. The data was sitting in the v2 scrape results we already had ‚Äî we just weren't extracting it.

**Diagnosis:** Domain doesn't tag floor plans separately (`category: "property"` for all images). Listing agents upload them in arbitrary positions ‚Äî most often near the end, but not reliably last. A 6-record visual sample showed 1/6 last images = floor plan; expanding to multi-listing aggregations (Legend Trail records with 100+ images), floor plans can be buried at position 121/161. Position heuristics fail; visual classification is required.

**Fix:** Built `scripts/extract_floor_plans_from_v2_images.py` ‚Äî GPT-4o-mini vision classifier with structured "YES/NO architectural floor plan?" prompt at `detail=low`. Per-record URL dedup; global thread-safe URL cache to skip API calls on shared images across records (saves ~5-15% of calls on duplicate-heavy datasets like golf-estate cohorts). Output written to non-destructive `floor_plans_v2_extracted[]` field + `floor_plans_v2_classifier` metadata (model, prompt version, per-candidate verdicts). Idempotent (skip if `floor_plans_v2_extracted_at` set; `--force` to override). Concurrent (`--workers` for records, `--image-workers` per record).

**Test (20 records, `--tail 3`):** 4/20 records found ‚â•1 floor plan (20% recall), 5 images extracted. **Precision: 5/5 manually verified = 100%.** Established the classifier works; recall capped by the last-3 search window.

- Eligible: 2,736 Robina records with ‚â•5 v2 images
- Runtime: 27.5 min (1,652s)
- Total images classified: 102,565 (5,762 cache hits = 5.6% saved)
