## Non-Target Suburb Coverage: Weekly Cadence Is By Design (Added 2026-03-28)
Non-target suburbs are scraped only on Sundays. Do not use non-target suburb staleness as a reason to limit trust claims for target suburbs (Robina, Burleigh Waters, Varsity Lakes), which are scraped daily and have separate freshness SLAs.
**Why it matters:** Growth messaging should not be restricted by non-target suburb freshness. Target suburbs have 12h SLA, non-targets have 7-day SLA.
**When to recall:** Any proposal about limiting trust language or coverage claims based on non-target suburb data.
